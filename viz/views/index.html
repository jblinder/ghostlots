About
For grassroots organizations, could predictive tools allow communities to engage with builders in community benefit agreements? Ghost Lots is a R&D project to see if urban infrastructural data can help communities anticipate imminent development in order to strategize preemptively.

(These predictions are entirely speculative and are soley based on the NYC PLUTO dataset.)

For more information, please visit https://jblinder.github.io.
